# 📊 Data Analysis: Brazilian E-Commerce Public Dataset (Olist)

## 📌 Deskripsi Project
Project ini merupakan analisis data end-to-end terhadap dataset Brazilian E-Commerce yang disediakan oleh Olist.  
Analisis dilakukan untuk memahami pola transaksi, perilaku pelanggan, serta performa produk dan wilayah penjualan.

Project ini dibuat sebagai bagian dari tugas UAS mata kuliah Analitik dan Manajemen Big Data.

---

## 🎯 Tujuan Analisis
- Mengidentifikasi kategori produk dengan performa terbaik
- Menganalisis distribusi pelanggan berdasarkan wilayah (state)
- Memahami hubungan antara nilai transaksi dengan rating pelanggan
- Menemukan insight yang dapat digunakan untuk pengambilan keputusan bisnis

---

## 📊 Dataset
- **Nama Dataset**: Brazilian E-Commerce Public Dataset by Olist  
- **Sumber**: Kaggle  
- **Link**: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce/data

Dataset ini mencakup berbagai tabel seperti:
- Data pelanggan (customers)
- Data pesanan (orders)
- Data produk (products)
- Data pembayaran (payments)
- Data review pelanggan (reviews)

## 🔄 Alur Analisis

### 1. Data Understanding
- Load dataset dari beberapa file CSV
- Memahami struktur dan tipe data

### 2. Data Quality Assessment
- Identifikasi missing values
- Pengecekan duplikasi data
- Analisis inkonsistensi data

### 3. Data Cleaning
- Penanganan missing values
- Penghapusan data yang tidak relevan
- Perbaikan format data

### 4. Data Integration
- Penggabungan beberapa tabel menggunakan join/merge
- Membentuk dataset utama untuk analisis

### 5. Feature Engineering
- Pembuatan fitur baru untuk mendukung analisis
- Transformasi data agar lebih informatif

### 6. Data Validation
- Validasi hasil cleaning dan integrasi
- Filtering data agar sesuai kebutuhan analisis

### 7. Exploratory Data Analysis (EDA) & Visualization
- Analisis distribusi data
- Visualisasi tren dan pola menggunakan grafik
- Interpretasi hasil analisis

---

## 📈 Insight Utama
Beberapa insight yang diperoleh dari analisis ini:

- Kategori produk tertentu memiliki jumlah transaksi yang jauh lebih tinggi dibandingkan kategori lainnya
- Distribusi pelanggan menunjukkan konsentrasi tinggi di beberapa state tertentu
- Terdapat hubungan antara nilai transaksi dan rating pelanggan, yang dapat mencerminkan tingkat kepuasan
- Pola pembelian pelanggan dapat digunakan untuk strategi pemasaran yang lebih efektif
